A sequence is called a Martin sequence if any term an (n > 3) of the sequence is of the following form: an = an - 1 + an - 2 + an - 3. Given the first three numbers of the sequence, you have to find the nth term.
The input contains the number of test cases T followed by T lines containing 4 numbers, a1, a2, a3 and n. You have to output an for every test case.
Given: 1 ≤ n ≤ 106, -1015 ≤ ai ≤ 1015

Sample Input

2
2 3 5 5
6 14 -1 8

Sample Output

18
183
