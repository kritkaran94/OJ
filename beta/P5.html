Problem 5

Given an integer N, check if it is a palindrome or not. If it is not a palindrome, print NO. If it is a palindrome, print YES. If it is an even palindrome, print the ratio of first half to second half, upto the precision of 2 decimal places. In cases where such a division results in an error print 0.00. If it is an odd palindrome, print the sum of the digits.
Given: 0 ≤ N ≤ 1015

Sample Input

1234321
89088098
780

Sample Output

YES
16
YES
1.10
NO
